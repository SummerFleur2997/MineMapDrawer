## 星露谷物语矿井地图渲染器

**简体中文 | [English](./README_en.md)**

一个轻量级的 TypeScript 工具，使用 HTML5 Canvas 渲染《星露谷物语》的矿井楼层布局。本项目特别针对 **MediaWiki** Widget 集成进行了优化。

### 功能特性

* **交互式渲染**：基于用户输入实时更新地图。
* **完整区域支持**：支持普通层、冰冻层、岩浆层、骷髅洞穴、史莱姆、史前层及采石场矿井层。
* **变体切换**：支持“黑暗”和“危险”（挑战之神龛）楼层变体。

### 安装与设置

有关本地开发或 MediaWiki 集成的详细说明，请参阅 [使用文档](./src/mediawiki%20widget/README.md)。

### C# 命令行版本

本项目还包含一个用于生成矿井地图的 C# CLI 工具。该程序可用于批量生成所有地图数据，详见 [使用文档](./src/csharp%20console%20app/README.md)。
