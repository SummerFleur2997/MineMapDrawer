## Stardew Valley Mine Map Drawer

**English | [简体中文](./README.md)**

A lightweight, TypeScript-based tool to render mine floor layouts from Stardew Valley using HTML5 Canvas.
This project is specifically optimized for integration into **MediaWiki** as a widget.

### Features

* **Interactive Rendering**: Real-time map updates based on user input.
* **Full Region Support**: Supports Normal, Frost, Lava, Desert, Slime, Dino, and Quarry floors.
* **Variant Toggles**: Includes "Dark" and "Dangerous" (Challenge Shrine) floor variants.

### Setup

For Local Development or MediaWiki Integration, please see the [doc](./src/mediawiki%20widget/README.md) (Chinese).

### C# CLI version

This project also includes a CLI tool for generating mine maps from C# code. Which can be used for generating
all the maps by running the program, see the [doc](./src/csharp%20console%20app/README.md) (Chinese) for more details.
